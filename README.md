# Counter App

A simple counter application built with HTML, CSS and JavaScript.

## Features
- Increase the counter
- Decrease the counter
- Reset the counter

## Purpose
This project was built while learning JavaScript by following a YouTube tutorial.

## Technologies Used
- HTML
- CSS
- JavaScript
